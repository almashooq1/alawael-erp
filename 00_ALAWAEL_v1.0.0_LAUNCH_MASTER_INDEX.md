## 🚨 WHAT IF SOMETHING GOES WRONG?

**In ALAWAEL_v1.0.0_FINAL_LAUNCH_PLAN.md**, under "PHASE 7: Rollback Procedure":

```bash
# Docker rollback
docker stop alawael-prod
docker run -d ... [previous-version]

# PM2 rollback
pm2 stop alawael
git checkout [previous-tag]
pm2 start ecosystem.config.js

# Systemd rollback
systemctl stop alawael
git checkout [previous-tag]
systemctl start alawael
```
