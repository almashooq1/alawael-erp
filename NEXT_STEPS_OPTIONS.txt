🔵 **الخيار 1️⃣ : تشغيل Frontend (React)**

الوصف:
  - تشغيل واجهة المستخدم الرسومية
  - اختبار التكامل مع Backend
  - رؤية النظام كاملاً في المتصفح

الخطوات:
  1. افتح Terminal جديد
  2. اذهب للمجلد: cd supply-chain-management/frontend
     أو: cd erp_new_system/frontend
  3. ثبت الحزم: npm install
  4. شغل الواجهة: npm start
  5. افتح المتصفح: http://localhost:3001

الوقت المتوقع: 10 دقائق ⏱️

المميزات:
✅ واجهة رسومية كاملة
✅ اختبار حقيقي للنظام
✅ تفاعل مباشر مع البيانات

═══════════════════════════════════════════════════════════════════

🔵 **الخيار 2️⃣ : اختبار API بشكل متقدم**

الوصف:
  - اختبار جميع Endpoints
  - التحقق من البيانات
  - اختبار الأخطاء والحالات الاستثنائية

الخطوات:
  1. استخدم Postman: https://postman.com/download
  2. أو استخدم Script: node test-api.js
  3. أو استخدم curl مع الأمثلة من API_TESTING_GUIDE.md

الملفات المساعدة:
  📖 API_TESTING_GUIDE.md
  📄 test-api.js
  📄 ERP_API_Postman_Collection.json

الوقت المتوقع: 15-20 دقيقة ⏱️
