⚡ أوامر سريعة جداً:

1. شغل Backend:
   ──────────────
   cd erp_new_system/backend
   npm install  # (تم بالفعل)
   npm start    # (يعمل الآن)

2. شغل Frontend (اختياري):
   ───────────────────────
   cd supply-chain-management/frontend
   npm install
   npm start

3. اختبر النظام:
   ─────────────
   الخيار A: المتصفح
   → http://localhost:3000

   الخيار B: Terminal
   → node test-api.js

   الخيار C: Postman
   → Base URL: http://localhost:3000

🚨 استكشاف المشاكل السريعة:

المشكلة: "Cannot find module..."
الحل: npm install

المشكلة: "Port 3000 is already in use"
الحل: taskkill /PID [PID] /F

المشكلة: "Connection refused"
الحل: تأكد من أن npm start يعمل

المشكلة: "API tidak merespons"
الحل: اقرأ السجلات في terminal

💡 نصائح مهمة:

🔹 Backend يعمل في الخلفية الآن
   - لا تغلق terminal حيث npm start يعمل
   - إذا أغلقت terminal، شغل npm start مجدداً

🔹 استخدم Postman للاختبار المتقدم
   - أسهل من curl
   - يحفظ الـ requests
   - يدعم Collections

🔹 اقرأ السجلات
   - افتح terminal Backend
   - ستشوف جميع requests والأخطاء

🔹 استخدم API_TESTING_GUIDE.md
   - يحتوي على أمثلة فعلية
   - يحتوي على حل المشاكل
