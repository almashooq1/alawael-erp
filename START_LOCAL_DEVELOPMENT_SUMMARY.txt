## 🆘 إذا حدثت مشكلة

### ❌ "Port already in use"
```bash
netstat -ano | findstr :3000
taskkill /PID <رقم> /F
```

### ❌ "npm install فشل"
```bash
npm cache clean --force
npm install
```

### ❌ "Node.js غير مثبت"
```
تحميل من: https://nodejs.org
```

اقرأ القسم الكامل في `QUICK_LOCAL_SETUP_GUIDE.md`
