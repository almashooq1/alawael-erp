# دليل المستخدم

## تسجيل الدخول
- أدخل اسم المستخدم وكلمة المرور حسب الدور (مدير، معلم، أخصائي، ولي أمر)

## لوحة التحكم
- إدارة الطلاب، الخطط، الجلسات، التقييمات، الأنشطة، الفريق، أولياء الأمور، التقارير، الوسائط
- عرض التحليلات الذكية والإشعارات

## بوابة ولي الأمر
- متابعة تقدم الطالب، الخطط، الجلسات، التقييمات، التقارير، الوسائط

## رفع الوسائط
- يمكن رفع صور، فيديو، صوت، مستندات وربطها بالطالب

## الإشعارات
- تظهر الإشعارات الجديدة في شريط العنوان ويمكن مراجعتها وتأكيد قراءتها

## الدعم
- لأي مشكلة أو استفسار: تواصل مع الدعم الفني
