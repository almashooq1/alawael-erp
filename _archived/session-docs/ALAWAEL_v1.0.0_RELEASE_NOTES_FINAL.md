## 🚀 **QUICK START GUIDE**

### Installation

```bash
# 1. Clone repository
git clone https://github.com/almashooq1/alawael-backend.git
cd alawael-backend

# 2. Checkout release version
git checkout v1.0.0

# 3. Install dependencies
npm install

# 4. Set up environment
cp .env.example .env
# Edit .env with your database URL, JWT secret, etc.

# 5. Start development server
npm start

# Server runs on http://localhost:3000
```

### Docker Deployment

```bash
# Build Docker image
docker build -t alawael-backend:1.0.0 .

# Run container
docker run -p 3000:3000 \
  -e DATABASE_URL=mongodb://localhost:27017/alawael \
  -e JWT_SECRET=your_secret_key \
  alawael-backend:1.0.0
```

### Verify Installation

```bash
# Health check endpoint
curl http://localhost:3000/api/health

# Get system statistics
curl http://localhost:3000/api/stats

# Run tests
npm test
```

## 📞 **SUPPORT & TROUBLESHOOTING**

### Common Issues & Solutions

**Issue: Database Connection Failed**
```bash
# Check MongoDB is running
mongosh

# Verify connection string in .env
cat .env | grep DATABASE_URL

# Test connection
node -e "const mongoose = require('mongoose'); mongoose.connect(process.env.DATABASE_URL)"
```

**Issue: Port Already in Use**
```bash
# Find and kill process
netstat -ano | findstr :3000
taskkill /PID <PID> /F

# Or change port in .env
PORT=3001 npm start
```

**Issue: Memory Issues**
```bash
# Check memory usage
node --max-old-space-size=4096 app.js

# Enable production mode
NODE_ENV=production npm start
```

**Issue: SSL Certificate Error**
```bash
# Verify certificate
openssl x509 -in cert.pem -text -noout

# Check certificate is valid for domain
openssl s_client -connect localhost:443
```
