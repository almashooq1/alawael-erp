# Contributing to Alawael ERP

مرحباً بكم في مشروع Alawael ERP! نسعد بمساهماتكم.

## 🚀 كيف تساهم

### 1. Fork المستودع
```bash
git clone https://github.com/almashooq1/alawael-erp.git
cd alawael-erp
```

### 2. إنشاء فرع جديد
```bash
git checkout -b feature/your-feature-name
```

### 3. تثبيت التبعيات
```bash
# Backend
cd backend
npm install

# Frontend
cd ../frontend
npm install
```

### 4. إجراء التغييرات
- اتبع معايير الكود الموجودة
- أضف اختبارات للميزات الجديدة
- تأكد من نجح جميع الاختبارات

### 5. تشغيل الاختبارات
```bash
npm test
npm run lint
```

### 6. إرسال Pull Request
- اشرح التغييرات بوضوح
- اربط بأي issues ذات صلة

## 📝 معايير الكود

### JavaScript/Node.js
- استخدم ES6+ features
- اتبع ESLint rules
- أضف تعليقات للدوال المعقدة

### Commit Messages
```
type(scope): description

# Examples:
feat(auth): add OTP authentication
fix(api): resolve connection timeout
docs(readme): update installation steps
```

## 🐛 الإبلاغ عن الأخطاء

1. تحقق من وجود الخطأ مسبقاً
2. استخدم قالب Issue
3. أضف خطوات إعادة الإنتاج

## 📧 التواصل
- GitHub Issues للأسئلة والأخطاء
- البريد الإلكتروني للمسائل الخاصة

شكراً لمساهمتكم! 🙏
