## ⚠️ سيناريوهات المخاطر

| المخاطر | التأثير على الجدول | الحل |
|--------|-------------------|------|
| **الموافقة التنفيذية متأخرة** | +5 إلى +10 أيام | Skip to MAR 8 (أسبوع واحد تأخير) |
| **AWS setup يأخذ وقت أطول** | +5 أيام حد أدنى | Parallel paths - monitoring بدون AWS |
| **مشاكل أمنية حرجة** | +7 إلى +14 يوم | Fix + NAG اختبار (لا يوجد خيار) |
| **فشل اختبار الحمل** | +3 إلى +7 أيام | Optimize code/infra (سريع جداً) |
| **UAT يرفع مشاكل جديدة** | +5 إلى +10 أيام | Priority fix + retest |
| **فريق رئيسي غير متاح** | +7 أيام ضياع | Activate backup + cross-training |
| **فشل أولي للنشر** | حسب السبب | Rollback فوري (إذا لزم الحال) |
