## 🎓 يوميات التعلم:

### نصائح للعمل مع النظام:

#### البيئة المحلية:

```bash
# تشغيل النظام محلياً
docker-compose up -d

# مشاهدة السجلات
docker-compose logs -f

# إيقاف النظام
docker-compose down
```

#### اختبار الوظائف:

```bash
# تشغيل الاختبارات
npm test

# اختبار الأداء
npm run benchmark

# التحقق من الجودة
npm run lint
```

#### استكشاف الأخطاء:

```bash
# عرض جميع الحاويات
docker ps

# فحص حاوية معينة
docker logs container-name

# الدخول إلى حاوية
docker exec -it container-name bash
```
