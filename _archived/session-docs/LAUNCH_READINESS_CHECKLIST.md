## 📋 خطوات الإطلاق الفوري

### الخطوة 1: إعداد البيئة
```bash
cd backend
cp .env.example .env
# تعبئة المتغيرات البيئية
```

### الخطوة 2: تثبيت الحزم
```bash
npm install
cd ../frontend
npm install
```

### الخطوة 3: بناء المشروع
```bash
npm run build
```

### الخطوة 4: النشر
```bash
# للنشر على Heroku:
heroku create alawael-erp
git push heroku main

# أو للنشر على VPS:
docker-compose up -d
```

## 🎯 الخيار الأسرع للإطلاق

### Heroku (5 دقائق)
```bash
# 1. تسجيل الدخول
heroku login

# 2. إنشاء التطبيق
heroku create alawael-erp-production

# 3. إعداد المتغيرات
heroku config:set NODE_ENV=production
heroku config:set MONGODB_URI=your_mongodb_uri
heroku config:set JWT_SECRET=your_secret

# 4. النشر
git push heroku main

# 5. فتح التطبيق
heroku open
```
