## 2. سياسة الخصوصية

### 2.1 البيانات المجمعة

| نوع البيانات | الغرض | مدة الاحتفاظ |
|--------------|-------|--------------|
| بيانات المستخدم الشخصية | إدارة الحساب | طوال فترة الحساب + 3 سنوات |
| سجلات النشاط | الأمان والمراقبة | 90 يوم |
| بيانات الجلسات | تجربة المستخدم | 30 يوم |
| ملفات تعريف الارتباط | التخصيص | 1 سنة |
| سجلات الدفع | الامتثال المالي | 7 سنوات |

### 2.2 حقوق المستخدم

```
يحق للمستخدم:
├─ الوصول لبياناته الشخصية
├─ تصحيح البيانات غير الدقيقة
├─ حذف بياناته (مع مراعاة القانون)
├─ نقل بياناته لجهة أخرى
├─ الاعتراض على المعالجة
└─ تقديم شكوى لجهة مختصة
```

### 2.3 مشاركة البيانات

```
لا يتم مشاركة البيانات مع أطراف ثالثة إلا:
├─ بموافقة صريحة من المستخدم
├─ لتنفيذ التزامات قانونية
├─ لحماية حقوق النظام
└─ لمقدمي الخدمات الموثوقين (تحت اتفاقية سرية)
```

## 3. شروط الخدمة

### 3.1 شروط الاستخدام

```
┌─────────────────────────────────────────────┐
│  TERMS OF SERVICE                           │
│  شروط خدمة منصة الأوائل                    │
├─────────────────────────────────────────────┤
│                                              │
│  1. قبول الشروط                             │
│  استخدام النظام يعني قبول هذه الشروط       │
│                                              │
│  2. الأهلية                                 │
│  ├─ يجب أن يكون المستخدم 18+               │
│  └─ أو لديه موافقة ولي الأمر               │
│                                              │
│  3. حساب المستخدم                           │
│  ├─ المسؤولية عن كلمة المرور               │
│  ├─ عدم مشاركة الحساب                      │
│  └─ الإبلاغ عن أي استخدام غير مصرح         │
│                                              │
│  4. الاستخدام المقبول                       │
│  ├─ الاستخدام للأغراض المشروعة فقط         │
│  ├─ عدم انتهاك حقوق الآخرين                │
│  └─ عدم محاولة اختراق النظام               │
│                                              │
│  5. التعديلات                               │
│  ├─ يحق للنظام تعديل الشروط                │
│  └─ يتم الإشعار قبل 30 يوم                 │
│                                              │
└─────────────────────────────────────────────┘
```

### 3.2 الأنشطة المحظورة

```
❌ ممنوع:
├─ محاولة الوصول غير المصرح به
├─ إدخال بيانات خاطئة عن قصد
├─ استخدام bots أو scrapers بدون إذن
├─ نشر محتوى ضار أو غير قانوني
├─ التسبب في ضغط غير طبيعي على النظام
├─ انتحال شخصية مستخدمين آخرين
└─ استخدام النظام لأغراض تجارية منافسة
```

### 3.3 الإخلاء من المسؤولية

```
النظام "كما هو" بدون ضمانات صريحة أو ضمنية.
لا يضمن النظام:
├─ عدم انقطاع الخدمة
├─ خلو النظام من الأخطاء
└─ دقة جميع المعلومات
```
