## 🆘 حل المشاكل السريعة

### مشكلة: "Connection refused"
```
الحل: تحقق من:
- عنوان المضيف الصحيح (IP address)
- المنفذ الصحيح (port) - عادة 3306
- اسم المستخدم وكلمة المرور
- أن قاعدة البيانات موجودة بالفعل
```

### مشكلة: "Unknown table"
```
الحل:
- قد لا تكون الجداول موجودة في قاعدة البيانات القديمة
- عدّل أسماء الجداول في الـ script ليطابق الواقع
```

### مشكلة: "Duplicate entry"
```
الحل:
- قد يكون هناك بيانات قديمة في قاعدة البيانات الجديدة
- نظّف قاعدة البيانات الجديدة أو احذفها وأعد إنشاؤها
```

### مشكلة: "Too many connections"
```
الحل:
- الانتظار قليلاً ثم إعادة المحاولة
- أو تقليل عدد الاتصالات المتزامنة في الـ config
```
