## 💻 كيفية استخدام النظام الآن

### 1️⃣ تشغيل Backend (يعمل بالفعل الآن)
```bash
cd erp_new_system/backend
npm install    # تم بالفعل
npm start      # يعمل الآن في الخلفية
```

### 2️⃣ اختبر في المتصفح
```
http://localhost:3000
http://localhost:3000/api/health
```

### 3️⃣ اختبر من Terminal
```bash
node test-api.js
```

### 4️⃣ تشغيل Frontend (اختياري)
```bash
cd supply-chain-management/frontend
npm install
npm start
```

### 5️⃣ استخدم Postman
- افتح Postman
- أضف Base URL: `http://localhost:3000`
- اختبر الـ endpoints المختلفة

## 🛠️ استكشاف الأخطاء

### المشكلة: "Connection Refused"
```bash
# تحقق من أن Backend يعمل
Get-Process | Where-Object {$_.ProcessName -like "*node*"}

# إذا لم يكن يعمل، شغله:
npm start
```

### المشكلة: Port 3000 مستخدمة
```bash
# ابحث عن العملية التي تستخدم المنفذ
netstat -ano | findstr :3000

# أيقفها إذا لزم الأمر
taskkill /PID [PID] /F
```

### المشكلة: حزم ناقصة
```bash
npm install       # شغل مجدداً
npm audit fix     # أصلح المشاكل الأمنية
```
