### 2️⃣ الذهاب إلى صفحة الإصدارات
- [ ] في الصفحة الرئيسية للمستودع، ابحث عن "Releases"
- [ ] اضغط على "Releases" أو: https://github.com/almashooq1/alawael-backend/releases
- [ ] اضغط على الزر "Draft a new release"

### 3️⃣ ملء نموذج الـ Release

#### حقل Tag
```

```
- [ ] أدخل: `v1.0.0` بدقة (كما هو مكتوب)
- [ ] تأكد من عدم وجود مسافات إضافية

#### حقل Release Title (العنوان)
```

```
- [ ] أدخل: `ALAWAEL v1.0.0 - Enterprise Platform`

#### حقل Description (الوصف)
- [ ] افتح ملف: **`GITHUB_RELEASES_DESCRIPTIONS.txt`**
- [ ] ابحث عن قسم: "🎯 المرحلة الأولى: ALAWAEL Backend"
- [ ] انسخ الوصف كاملاً (البدء من "🎉" إلى "**آخر تحديث**")
- [ ] الصقه في حقل Description

### 4️⃣ خيارات إضافية (اختياري)
- [ ] **This is a pre-release**: اتركها غير مختارة (✗)
- [ ] **Create a discussion**: يمكنك اختيارها (✓)
- [ ] **Set as the latest release**: اخترها (✓) لجعلها الإصدارة الأخيرة

### 5️⃣ النشر
- [ ] اضغط الزر الأخضر: **"Publish release"**
- [ ] انتظر تحميل الصفحة

### 6️⃣ التحقق من النجاح
```
✓ يجب أن ترى رسالة تأكيد
✓ يجب أن تظهر v1.0.0 في القائمة
✓ يجب أن تظهر صورة الإصدار
```
- [ ] تحقق من ظهور `v1.0.0` في قائمة الإصدارات
- [ ] انقر عليها للتحقق من صحة البيانات
- [ ] تأكد من ظهور الوصف الكامل
