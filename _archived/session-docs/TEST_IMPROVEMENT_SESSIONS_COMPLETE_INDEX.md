## 🛠️ Useful Command Reference

### Running Tests
```bash
# All tests
npm test

# Specific project
cd intelligent-agent && npm test
cd backend && npm test
cd supply-chain-management && npm test

# With coverage
npm test -- --coverage

# Watch mode
npm test -- --watch

# Specific file
npm test tests/user-management.test.ts

# With pattern
npm test -- --testNamePattern="specific test"
```

### Checking Status
```bash
# Test results summary
npm test 2>&1 | grep "Tests:"

# Coverage report
npm test -- --coverage --coverageReporters=text-summary

# Performance metrics
npm test 2>&1 | grep "Duration:"
```
