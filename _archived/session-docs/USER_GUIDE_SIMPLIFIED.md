## 2. المهام الشائعة

### 👥 إدارة المستخدمين

| المهمة | الخطوات |
|--------|---------|
| إضافة مستخدم | المستخدمون → إضافة →填写 البيانات → حفظ |
| تعديل مستخدم | المستخدمون → بحث → تحديد → تعديل → حفظ |
| حذف مستخدم | المستخدمون → بحث → تحديد → حذف → تأكيد |

### 📄 إدارة المستندات

| المهمة | الخطوات |
|--------|---------|
| رفع مستند | المستندات → رفع → اختيار الملف → رفع |
| البحث عن مستند | المستندات → بحث → كتابة الكلمة → بحث |
| تحميل مستند | المستندات → تحديد → تحميل |
