## 📋 خطوات التشغيل

### الطريقة الأولى: تشغيل تلقائي
```bash
# في نافذة终端 1 - تشغيل Backend
node run-backend.js

# في نافذةTerminal 2 - تشغيل Frontend
cd frontend && npm start
```

### الطريقة الثانية: ملفات BAT
1. انقر مرتين على `start-backend.bat` لتشغيل Backend
2. افتح نافذة جديدة وشغل:
   ```bash
   cd frontend && npm start
   ```

## 🛠️ استكشاف الأخطاء

### المشكلة: Backend لا يعمل
```bash
# تحقق من MongoDB
mongod --version

# تشغيل MongoDB (إذا لم تكن تعمل)
mongod
```

### المشكلة: Frontend لا يتصل بـ Backend
- تأكد أن Backend يعمل على المنفذ 3001
- تحقق من ملف `.env` في Frontend

### المشكلة: خطأ في قاعدة البيانات
- تأكد من وجود ملف `.env` في مجلد backend
- تحقق من إعدادات MONGODB_URI
