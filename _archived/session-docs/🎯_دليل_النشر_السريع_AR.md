## 🚀 خيارات النشر (اختر واحد)

### **الخيار أ: Docker** ⭐ *موصى به* (15 دقيقة)

**الأفضل لـ**: النشر السريع، الحاويات، التوسع التلقائي

```bash
# الخطوة 1: الانتقال إلى مجلد الخادم
cd dashboard/server

# الخطوة 2: نسخ ملف البيئة
cp .env.example .env.production

# الخطوة 3: تعديل .env.production
# ضع عناوين قاعدة البيانات و Redis الخاصة بك
# DATABASE_URL=postgresql://user:password@host:5432/alawael_erp
# REDIS_URL=redis://host:6379
# NODE_ENV=production

# الخطوة 4: بناء صورة Docker
docker build -t alawael-api:v1.0.0 .

# الخطوة 5: تشغيل الحاوية
docker run -d \
  --name alawael-prod \
  -p 3001:3001 \
  --env-file .env.production \
  --restart unless-stopped \
  alawael-api:v1.0.0

# الخطوة 6: التحقق من التشغيل
curl http://localhost:3001/health
```

**متى تستخدمه**: تريد نشراً سريعاً بالحاويات، تخطط للتوسع لاحقاً مع Kubernetes
