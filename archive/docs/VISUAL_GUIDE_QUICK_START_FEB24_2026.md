## 🚨 Common Mistakes to Avoid

❌ Don't:
- Push without adding GitHub Secret
- Push without running local tests first
- Forget to chmod +x the scripts
- Skip the aws-setup.sh script
- Push to wrong branch (use main, not master)

✅ Do:
- Test locally first
- Add the AWS_ACCOUNT_ID secret
- Make scripts executable
- Run aws-setup.sh before push
- Double-check branch is main
