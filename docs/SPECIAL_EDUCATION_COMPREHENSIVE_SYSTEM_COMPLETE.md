## 🧩 برامج التربية الخاصة المدعومة

| البرنامج | نوع الإعاقة | الاستراتيجيات |
|----------|-------------|---------------|
| 🧩 التوحد | `autism` | ABA, TEACCH, PECS, VBA |
| 📚 صعوبات التعلم | `learning_disability` | متعدد الحواس، أورتن جيلينغهام |
| 🧠 الإعاقة الذهنية | `intellectual_disability` | التعليم المباشر، التعلم التجريبي |
| ⚡ فرط الحركة | `adhd` | التعليم المتمايز، إدارة السلوك |
| 👂 الإعاقة السمعية | `hearing_impairment` | لغة الإشارة، القراءة الشفوية |
| 👁️ الإعاقة البصرية | `visual_impairment` | برايل، الصوتيات، اللمسية |
| 🔄 الإعاقة المتعددة | `multiple_disabilities` | التعليم الفردي، التقنيات المساعدة |
