## 📊 نظام متابعة الحضور

### أنواع الحالة:
| الحالة | الوصف |
|--------|-------|
| `present` | حاضر |
| `absent` | غائب |
| `late` | متأخر |
| `excused` | غياب بعذر |
| `early_dismissal` | انصراف مبكر |

### الإحصائيات المتتبعة:
- معدل الحضور الكلي
- معدل الالتزام بالمواعيد
- الغياب المتتالي
- تحليل أنماط الغياب

## 🔄 نظام متابعة السلوك

### أنواع السلوك:
| النوع | الوصف |
|-------|-------|
| `positive` | إيجابي - للتعزيز |
| `negative` | سلبي - للتعديل |
| `neutral` | محايد - للملاحظة |

### تصنيفات السلوك:
- 📚 أكاديمي
- 👥 اجتماعي
- 🔄 سلوكي
- 💭 عاطفي

### مستويات الشدة:
| المستوى | الوصف | الإجراء |
|---------|-------|---------|
| `minor` | بسيط | توثيق |
| `moderate` | متوسط | توثيق + تدخل |
| `major` | كبير | إبلاغ + خطة |
| `critical` | حرج | إجراءات طارئة |
