## ⚠️ الأخطاء الشائعة

### مشكلة: الزر "Draft a new release" لا يظهر
**السبب:** لم تسجل الدخول  
**الحل:** تأكد من تسجيل دخول GitHub

### مشكلة: الوصف لا يظهر بشكل صحيح
**السبب:** تم نسخ نص ناقص  
**الحل:** انسخ من الملف كاملاً بدون حذف

### مشكلة: الـ Tag رفضوه
**السبب:** صيغة خاطئة (مثل `1.0.0` بدون `v`)  
**الحل:** استخدم بالضبط: `v1.0.0`

### مشكلة: الصفحة لم تحدّث بعد النشر
**السبب:** تأخر التحديث  
**الحل:** اضغط F5 أو أعد تحميل الصفحة
