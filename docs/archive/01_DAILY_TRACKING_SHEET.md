## 🆘 **الإجراءات الطارئة القرارات السريعة**

### **إذا حدثت مشكلة في أي وقت:**

```
ISSUE: خطأ في النقل
  → أوقف الهجرة فوراً
  → اتصل بـ DBA + VP
  → القرار: تصحيح (30 دق) أم rollback؟
  → Run: node migration-rollback.js

ISSUE: مساحة تخزين:
  → تنبيه فوري
  → احذف الملفات المؤقتة
  → أعد المحاولة

ISSUE: اتصال قطع:
  → إعادة الاتصال
  → جرب من جديد
  → إذا استمر → rollback

ISSUE: بيانات فاسدة:
  → توثيق دقيق للمشكلة
  → اتصل بـ CTO
  → التقييم والقرار
```
