## 🔧 كيفية الاستخدام

### البدء السريع

```javascript
// 1. استيراد الخدمة
const AdvancedArchivingSystem = require('./services/advancedArchivingSystem');

// 2. إنشاء مثيل
const archiving = new AdvancedArchivingSystem();

// 3. أرشفة مستند
const result = await archiving.archiveDocument({
  name: 'فاتورة_يناير.pdf',
  content: 'محتوى الفاتورة',
  type: 'application/pdf',
});

// 4. البحث
const results = archiving.smartSearch('فاتورة');

// 5. الاسترجاع
const archive = await archiving.retrieveArchive(result.archiveId);
```
