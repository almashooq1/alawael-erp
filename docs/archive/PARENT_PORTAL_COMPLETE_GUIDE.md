# 🎊 بوابة الآباء والأولياء - دليل شامل

## نظرة عامة

بوابة الآباء والأولياء هي نظام متقدم يسمح لأولياء الأمور بمتابعة تقدم أطفالهم، التواصل مع المعالجين، وإدارة جميع جوانب الخدمات الطبية والتعليمية.

---

## 🏆 الميزات الرئيسية

### 1. **لوحة المعلومات الرئيسية (ParentDashboard)**

#### المحتوى:

- **اختيار الطفل**: بطاقات للاختيار بين عدة أطفال
- **بطاقة معلومات الطفل**:

  - الاسم والعمر
  - التقدم الإجمالي (نسبة مئوية)
  - نسبة الحضور
  - عدد الجلسات المكتملة
  - عدد الأيام للجلسة القادمة

- **علامات تبويب ديناميكية**:

  1. **التقدم والمهارات**: عرض المهارات الأساسية مع أشرطة التقدم
  2. **الجلسات القادمة**: جدول بالجلسات المقادمة
  3. **التواصل**: بطاقات المعالجين مع معلومات التواصل
  4. **المستندات**: قائمة بأحدث المستندات

- **قسم التنبيهات**: إشعارات مهمة من المركز والمعالجين

#### التصميم:

- رأس متدرج أرجواني (#667eea → #764ba2)
- بطاقات ملونة للإحصائيات
- جداول وعلامات تبويب تفاعلية
- تحديثات فورية للبيانات

---

### 2. **تتبع التقدم والمهارات (ChildrenProgress)**

#### المحتوى:

- **بطاقات الملخص**:

  - عدد المهارات المحسنة
  - متوسط التقدم
  - عدد العلامات الفارقة المحققة

- **علامات تبويب متعددة**:

  1. **المهارات الرئيسية**: عرض مفصل لكل مهارة

     - اسم المهارة
     - نسبة التقدم (رسم بياني ملون)
     - الهدف النهائي
     - المعالج المسؤول
     - تاريخ آخر تحديث

  2. **التطور الشهري**: جدول يوضح التقدم شهرياً

     - الشهر
     - نسبة التقدم
     - عدد الجلسات
     - ملاحظات

  3. **الإنجازات**: بطاقات ملونة للإنجازات المحققة

     - عنوان الإنجاز
     - التاريخ
     - تصميم متدرج جميل

  4. **النقاط المحسنة**: قائمة بالتحسينات الملحوظة
     - الاسم والوصف
     - نسبة التحسن
     - التاريخ

#### التصميم:

- رأس متدرج أخضر (#43e97b → #38f9d7)
- رسوم بيانية بألوان متعددة تعكس التقدم
- بطاقات متجاوبة

---

### 3. **تقارير الحضور والسلوك (AttendanceReports)**

#### المحتوى:

- **إحصائيات الملخص**:

  - إجمالي الجلسات المكتملة
  - نسبة الحضور
  - عدد الغيابات
  - عدد المتأخرات

- **أدوات التصفية**:

  - اختيار الشهر
  - نوع التقرير (حضور/سلوك/أداء)
  - زر الطباعة

- **علامات تبويب المحتوى**:

  1. **تقرير الحضور**: جدول مفصل بكل جلسة

     - التاريخ والوقت
     - اسم المعالج
     - حالة الحضور (حاضر/غياب/متأخر) - رموز ملونة
     - ملاحظات

  2. **تقرير السلوك**: بطاقات توضح السلوكيات

     - السلوكيات الإيجابية (شرائح خضراء)
     - نقاط تحتاج التحسين (شرائح حمراء)
     - ملخص سلوكي

  3. **تقرير الأداء**: أشرطة التقدم للمؤشرات
     - مستوى الاستيعاب
     - التواصل والتفاعل
     - الالتزام والانضباط
     - المهارات الحركية

#### التصميم:

- رأس متدرج وردي (#fa709a → #fee140)
- جداول محترفة مع ألوان حالة مختلفة
- أشرطة تقدم متدرجة

#### المحتوى:

  - حقل الإدخال مع أيقونات الملفات والإرسال

- **الإجراءات السريعة**:
  - جدولة مكالمة
  - إرسال استفسار
  - إرسال ملف
  - طلب استشارة

  - حقل التاريخ
  - حقل الوقت

  - إعلانات مهمة
  - نقاشات فعالة

- **علامات التبويب الثلاث**:

- **ديالوج الرسالة الجديدة**:
