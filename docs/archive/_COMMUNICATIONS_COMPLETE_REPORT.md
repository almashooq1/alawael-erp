```javascript
communicationsAPI = {
  getDashboardData(),           // جلب بيانات لوحة المعلومات

  getConversationMessages(),    // جلب رسائل محادثة
  chatWithBot(),                // محادثة مع البوت
  getEmails(),                  // جلب البريد
  sendEmail(),                  // إرسال بريد
  getAnalytics()                // جلب التحليلات
}
```

GET    /api/ai-communications/emails?folder=inbox

### سيناريوهات الاختبار

#### اختبار المساعد الذكي

```text
1. انقر على تبويب "المساعد الذكي"
2. اكتب: "كيف أضيف طالب جديد؟"
3. اضغط Enter أو زر الإرسال
4. تحقق من الرد الذكي خلال ثانية
5. جرب الاقتراحات السريعة
```

#### اختبار الرسائل المباشرة

```text
1. انقر على تبويب "الرسائل"
2. اختر محادثة من القائمة
3. اكتب رسالة جديدة
4. تحقق من ظهورها فوراً
5. جرب البحث في المحادثات
```

#### اختبار البريد الداخلي

```text
1. انقر على تبويب "البريد الداخلي"
2. افتح رسالة من "البريد الوارد"
3. انقر "إنشاء بريد جديد"
4. املأ البيانات (إلى، موضوع، رسالة)
5. حدد الأولوية (عاجل/عادي/منخفض)
6. اضغط "إرسال"
```

#### اختبار التحليلات

```text
1. انقر على تبويب "التحليلات"
2. تصفح الرسوم البيانية
3. مرر فوق النقاط لعرض التفاصيل
4. تحقق من الإحصائيات الشاملة
```
