## 📤 رفع الوثائق | Uploading Documents

### الخطوة 1: الانتقال إلى قسم الوثائق | Step 1: Navigate to Documents

```text
القائمة الجانبية → الوثائق → إضافة وثيقة جديدة
Sidebar Menu → Documents → Add New Document
```

أو اضغط على الزر الأحمر:

```text
[+ رفع وثيقة جديدة | + Upload New Document]
```

### الخطوة 2: اختيار الملف | Step 2: Choose File

```text
[📎 اختر الملف | Choose File]
```

**الملفات المدعومة | Supported File Types:**

- 📄 المستندات: PDF, DOC, DOCX, TXT
- 📊 الجداول: XLS, XLSX, CSV
- 📸 الصور: JPG, PNG, GIF, BMP
- 🎥 الفيديو: MP4, AVI (حسب الإعدادات)
- 📦 الأرشيفات: ZIP, RAR (إذا لزم الأمر)

**الحد الأقصى لحجم الملف:** 50 MB

### الخطوة 3: ملء البيانات الوصفية | Step 3: Fill Metadata

| الحقل       | المثال                        | مطلوب |
| ----------- | ----------------------------- | ----- |
| **العنوان** | عقد البيع                     | ✅    |
| **الوصف**   | عقد بيع مركبة رقم VH-2025-001 | ⭕    |
| **الفئة**   | عقود، تقارير، إلخ             | ✅    |
| **الوسوم**  | مهم، سري، عاجل                | ⭕    |

### الخطوة 4: حفظ الوثيقة | Step 4: Save Document

```text
[✅ حفظ وتحميل | Save & Upload]
```

سترى التقدم:

```text
جاري التحميل... [████████░░░░] 67%
Uploading... [████████░░░░] 67%
```

عند الانتهاء:

```text
✅ تم رفع الوثيقة بنجاح
✅ Document uploaded successfully
معرف الوثيقة: DOC-2025-00042
```

## 🔍 البحث والتصفية المتقدمة | Advanced Search & Filtering

### البحث السريع | Quick Search

```text
[🔍 ابحث عن وثيقة... | Search documents...]
```

**أمثلة على البحث:**

- `عقد` → يعرض جميع العقود
- `تقرير طبي` → يعرض التقارير الطبية
- `أحمد محمد` → يعرض وثائق العميل أحمد

### التصفية المتقدمة | Advanced Filters

اضغط على **[🔽 فلاتر | Filters]** لعرض الخيارات:

#### 1️⃣ التصفية حسب التاريخ | By Date

```text
من | From: [📅 تاريخ]
إلى | To:   [📅 تاريخ]

أمثلة سريعة | Quick Presets:
├─ اليوم | Today
├─ هذا الأسبوع | This Week
├─ هذا الشهر | This Month
└─ هذه السنة | This Year
```

#### 2️⃣ التصفية حسب حجم الملف | By File Size

```text
الحد الأدنى | Min: [___] KB
الحد الأقصى | Max: [___] KB

أمثلة سريعة:
├─ صغير (< 1 MB) | Small
├─ متوسط (1-10 MB) | Medium
└─ كبير (> 10 MB) | Large
```

#### 3️⃣ التصفية حسب الفئة | By Category

```text
☑ عقود | Contracts
☑ تقارير | Reports
☑ سياسات | Policies
☑ تدريب | Training
☑ مالي | Financial
☐ أخرى | Other
```

#### 4️⃣ التصفية حسب الوسوم | By Tags

```text
[🏷️ اختر الوسوم | Select Tags]

أمثلة على الوسوم:
├─ مهم | Important
├─ سري | Confidential
├─ عاجل | Urgent
├─ موافق | Approved
└─ قيد المراجعة | Under Review
```

### الترتيب والتنسيق | Sorting & Layout

**ترتيب حسب | Sort By:**

```text
[▼ الترتيب]
├─ العنوان (أبجدي) | Title (A-Z)
├─ الفئة | Category
├─ حجم الملف | File Size
└─ التاريخ (الأحدث أولاً) | Date (Newest First)
```

**تخطيط العرض | Layout:**

```text
[☷ قائمة | List View] [☶ شبكة | Grid View]
```

## ✏️ تحرير الوثائق | Editing Documents

### تعديل البيانات الوصفية | Edit Metadata

1. افتح الوثيقة
2. اضغط **[✏️ تعديل | Edit]**
3. عدّل البيانات:
   - العنوان
   - الوصف
   - الفئة
   - الوسوم
4. اضغط **[✅ حفظ التغييرات | Save Changes]**

**مثال على التعديل:**

```text
العنوان القديم: عقد_البيع
العنوان الجديد: عقد بيع مركبة - تويوتا ياريس 2025

الوسوم الجديدة: مهم، موافق من المدير، عاجل
```

### تغيير الفئة | Change Category

```text
الفئة القديمة: ملفات أخرى
الفئة الجديدة: عقود

✅ الفئة المحدثة | Category updated
```

## 📤 تصدير البيانات | Exporting Data

### تصدير إلى CSV | Export to CSV

**الخطوة 1:** اختر الوثائق المراد تصديرها

```text
☑ حدّد جميع الوثائق المرئية
☑ Select all visible documents
أو قم بتحديد فردي | or select individually
```

**الخطوة 2:** اضغط على **[📥 تصدير CSV | Export to CSV]**

**النتيجة:** ملف CSV يحتوي على:

```text
العنوان,الفئة,الحجم,التاريخ,الوسوم,عدد المشاهدات
Title,Category,Size,Date,Tags,Views
عقد البيع,عقود,2.3MB,2026-01-14,"مهم, موافق",42
تقرير المبيعات,تقارير,1.1MB,2026-01-13,"تقرير, شهري",28
```

### تصدير إلى JSON | Export to JSON

**الخطوة 1:** اختر الوثائق
**الخطوة 2:** اضغط على **[📥 تصدير JSON | Export to JSON]**

**النتيجة:** ملف JSON بالبيانات الكاملة:

```json
{
  "documents": [
    {
      "id": "DOC-2025-00042",
      "title": "عقد البيع",
      "category": "عقود",
      "size": 2400,
      "dateCreated": "2026-01-14T10:30:00Z",
      "tags": ["مهم", "موافق"],
      "uploadedBy": "Dr. Ahmed Mohammed"
    }
  ]
}
```

## 🔄 العمليات الجماعية | Bulk Operations

### تحديد الوثائق | Select Documents

**طريقة 1: تحديد فردي**

```text
اضغط على ☑ بجانب الوثيقة
Click ☑ next to document
```

**طريقة 2: تحديد الصفحة الحالية**

```text
☑ في رأس الجدول = تحديد جميع الوثائق في الصفحة
☑ in table header = select all on this page
```

**طريقة 3: تحديد جميع النتائج المصفاة**

```text
اختر من القائمة → [تحديد كل النتائج | Select All Results]
⚠️ تحذير: إذا كان > 100 وثيقة ستظهر موافقة
⚠️ Warning: Confirmation dialog if > 100 documents
```

### العمليات المتاحة | Available Operations

#### 1️⃣ تحميل جماعي | Bulk Download

```text
[عدد الوثائق المحددة: 5]
[📥 تحميل المحدد | Download Selected]

سيتم إنشاء ملف ZIP يحتوي على:
├─ عقد البيع.pdf
├─ تقرير المبيعات.xlsx
├─ صورة_المركبة.jpg
├─ استمارة_موافقة.docx
└─ وثيقة_سرية.pdf
```

#### 2️⃣ حذف جماعي | Bulk Delete

```text
[🗑️ حذف المحدد | Delete Selected]

تحذير | Warning:
هل أنت متأكد من رغبتك في حذف 5 وثائق؟
Are you sure you want to delete 5 documents?

[❌ إلغاء | Cancel] [✅ أؤكد الحذف | Confirm]
```

#### 3️⃣ تعديل الوسوم جماعي | Bulk Edit Tags

```text
[🏷️ تحرير الوسوم | Edit Tags]

الوسوم الحالية: مهم, موافق
إضافة وسوم جديدة:

[___] أدخل الوسوم مفصولة بفواصل

أمثلة: سري، عاجل، قيد المراجعة

[✅ تطبيق | Apply]
```

#### 4️⃣ تغيير الفئة جماعي | Bulk Change Category

```text
[📁 تغيير الفئة | Change Category]

├─ عقود | Contracts

├─ تدريب | Training
└─ مالي | Financial
```

#### 5️⃣ مشاركة جماعية | Bulk Share

```text

إرسال الوثائق إلى:
```

## 💡 أفضل الممارسات | Best Practices

### ✅ ما يجب فعله | DO's

**استخدم أسماء واضحة:**

```text
✓ "عقد بيع مركبة تويوتا ياريس 2025 - أحمد محمد"
✓ "تقرير المبيعات الشهري - يناير 2026"
```

**استخدم الفئات والوسوم:**

```text
الفئة: عقود
الوسوم: مهم, موافق, قانوني
```

**نظّم الملفات بانتظام:**

- قم بمراجعة الملفات غير المستخدمة
- احذف النسخ المكررة
- حديّث البيانات الوصفية

### ❌ ما يجب تجنبه | DON'Ts

**أسماء غير واضحة:**

```text
✗ "ملف.pdf"
✗ "وثيقة_جديدة"
✗ "copy_of_copy.docx"
```

**عدم تنظيم الملفات:**

```text
✗ لا تترك جميع الملفات بدون تصنيف
✗ لا تستخدم وسوم عشوائية
```

**الملفات الكبيرة جداً:**

```text
✗ تجنب الملفات > 50 MB
✗ ضغط الصور قبل الرفع
```
