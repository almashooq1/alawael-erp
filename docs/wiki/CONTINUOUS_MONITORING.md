## 🆘 نظام المساعدة الفوري

### إذا حدثت مشكلة:

#### ❌ "npm: command not found"

```bash
# الحل:
Set-ExecutionPolicy RemoteSigned -Scope CurrentUser -Force
# ثم أعد تشغيل Terminal
```

#### ❌ "Port already in use"

```bash
# الحل:
# استخدم port مختلف:
PORT=3002 npm start
```

#### ❌ "Cannot find module"

```bash
# الحل:
rm -r node_modules
npm install
```

#### ❌ "Dependencies conflict"

```bash
# الحل:
npm cache clean --force
npm install
```
