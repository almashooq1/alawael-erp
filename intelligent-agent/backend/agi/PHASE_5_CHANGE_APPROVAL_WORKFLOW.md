## 4. سير العمل | Workflow Steps

1. **طلب تغيير**: تعبئة نموذج طلب التغيير
2. **مراجعة مبدئية**: Change Manager + Ops
3. **تقييم المخاطر**: Security + QA
4. **اختبار قبل التغيير**: QA validates baseline
5. **الموافقة النهائية**: Change Manager approves
6. **تنفيذ التغيير**: خلال نافذة التغيير
7. **اختبار بعد التغيير**: Verify KPIs
8. **إغلاق الطلب**: توثيق النتائج والدروس المستفادة
