## 📍 كيفية البدء (الخطوات الفورية)

### **الخيار 1: تشغيل مع Docker (الأفضل)**

```bash
cd supply-chain-management
docker-compose up -d

# ثم الوصول إلى:
# الواجهة الأمامية: http://localhost:3000
# الـ API الخلفي: http://localhost:4000
# وثائق ال API: http://localhost:4000/api/docs
```

### **الخيار 2: تشغيل الاختبارات فقط**

```bash
# اختبارات Backend (جميعها نجحت)
cd backend
npm test

# اختبارات Frontend
cd ../frontend
npm test
```

### **الخيار 3: التطوير المحلي**

```bash
# المحطة الأولى: Backend
cd backend
npm install
npm start

# المحطة الثانية: Frontend
cd frontend
npm install
npm start

# MongoDB يجب أن يكون قيد التشغيل (محليًا أو عبر Docker)
```
