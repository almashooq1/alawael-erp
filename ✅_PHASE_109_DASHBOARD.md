## 1. Overview

The **Digital Twin Dashboard** provides a visual interface for the "Smart" ecosystem. It moves the project from JSON-only API interactions to a user-facing clinical tool.

## 2. Key Features

- **Visualized Holistic Score:** A dynamic gauge showing the patient's overall health status (0-100).
- **Layered Data Cards:** Distinct visual sections for Physical, Mental, Metabolic, and Predictive data.
- **Real-Time Integration:** Fetches data directly from `smartPatientIntegrator` (Phase 108).
- **AI Recommendation Box:** Highlighted actionable insights generated by the system.

## 3. Technical Implementation

- **Frontend Stack:** HTML5, CSS3, Vanilla JS (No heavy framework required for this lightweight dashboard).
- **Location:** `frontend_smart/`
  - `dashboard.html`: Structure.
  - `dashboard.js`: Fetch logic & DOM manipulation.
- **Server Integration:** Served statically by `server_smart.js` at `/dashboard`.

## 4. How to Access

1. Start the Smart Server: `node backend/server_smart.js` (or use the PS1 wrapper).
2. Open Browser: `http://localhost:3001/dashboard/dashboard.html`.
3. Input Patient ID (Default: `TEST_DIGITAL_TWIN_01`).
4. View the generated Digital Twin.
